# Campus Connect 🎓📱

A full-stack campus ecosystem built with React Native, Expo, TypeScript, and Supabase.

Campus Connect is a production-style mobile application that centralizes student life into a single platform. Instead of requiring separate systems for events, communication, academics, transportation, dining, career services, and student engagement, Campus Connect provides a unified mobile experience with real-time updates, secure authentication, and scalable backend architecture.

### Project Highlights

* Built with **React Native + Expo** for cross-platform mobile development
* Developed a **full-stack architecture** using Supabase (PostgreSQL, Authentication, Realtime, Storage)
* Implemented **real-time messaging and community features**
* Designed and structured a large-scale application with **15+ integrated modules**
* Utilized **TypeScript** for type safety and maintainability
* Implemented **Row Level Security (RLS)** and secure authentication workflows
* Created a scalable database schema supporting users, events, messaging, academics, finances, and campus services
